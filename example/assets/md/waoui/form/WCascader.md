# 级联选择器

```widget
cascaderOptions
```

## 基本用法
```widget
cascaderBasic
```

## 禁用选项
```widget
cascaderDisabled
```

## 可清除
```widget
cascaderClear
```

## 只显示最子级
```widget
cascaderLastOne
```

## 多选
### 显示所有Tag
```widget
cascaderMultiSelect1
```
### 只保留一个选项，其他选项以数字代替
```widget
cascaderMultiSelect2
```


## 任一级别均可选择
### 单选
```widget
cascaderAnyOne1
```
### 多选
```widget
cascaderAnyOne2
```

## 动态加载
```widget
cascaderDynamicLoad
```

## 可搜索
### 单选
```widget
cascaderSeach1
```
### 多选
```widget
cascaderSeach2
```

## 自定义节点
```widget
cascaderCustomNode
```

## 仅使用面板
```widget
cascaderPanel
```
