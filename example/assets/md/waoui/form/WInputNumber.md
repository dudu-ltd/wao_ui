# 数字输入框
## 基本用法
```widget
inputNumberBasic
```

## 禁用
```widget
inputNumberDisabled
```

## 固定增减步长
```widget
inputNumberStep
```

## 严格步长
```widget
inputNumberStepStrictly
```

## 保留小数位
```widget
inputNumberPrecision
```

## 不同尺寸
```widgets
inputNumberSize1,inputNumberSize2
```
```widgets
inputNumberSize3,inputNumberSize4
```

## 按钮位置
```widget
inputNumberBtnPosition
```
