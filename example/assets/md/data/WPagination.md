# 分页

## 基本用法
```widget
paginationBasicFew
```
## 省略页码
```widget
paginationBasicLot
```

## 简单模式
```widget
paginationSimple
```

## 设置快速选择的页码数量为 11 个
```widget
paginationButtons
```


## 显示背景色
```widget
paginationBacgroudColor
```

## 尺寸 
```widget
paginationSmall
```

## 附加功能
### 显示总数
```widget
paginationShowTotal
```

### 调整每页显示条数
```widget
paginationChangeSize
```


### 直接前往

```widget
paginationJumper
```

### 完整功能

```widget
paginationFull
```
