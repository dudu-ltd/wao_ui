# 下拉菜单
## 基本用法
```widget
dropdownBasic
```

## 下箭头与按钮分离
```widgets
dropdownSplit1,dropdownSplit2
```

## 触发方式
```widgets
dropdownTriggerClick,dropdownTriggerHover
```

## 禁用
```widget
dropdownDisabled
```

## 命令组
```widget
dropdownCommand
```

## 不同尺寸
```widgets
dropdownSize1,dropdownSize2
```
```widgets
dropdownSize3,dropdownSize4
```
